INSTALLATION

Follow these steps to load it into your browser manually:

1. Download the .zip file from the Google Drive link.

2. Extract the contents into a permanent folder on your computer.

3. Open Google Chrome and navigate to:
   chrome://extensions/

4. Turn on "Developer mode" using the toggle in the top-right corner.

5. Click the "Load unpacked" button.

6. Select the extracted folder containing the manifest.json file.

From now on, every time you open a new tab, Origin New Tab will load automatically.

PRIVACY

This extension does NOT collect, store, or transmit any personal data.
No browsing data, location data, or usage data is saved or sent anywhere.
Everything runs locally on your device.
